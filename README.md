# AI-Chat-Mediator
Project for mediating language gaps in real-time conversations.

Initial setup is as follows:

1. Within the root level of the gptBackend folder (where manage.py is located), created a new '.env' file wherein storing your OpenAI api key.
- `OPENAI_API_KEY = your key`

2. Initialize a Vue project called 'chat-gpt', and then replace the 'public' and 'src' folders with those in the frontend folder.
